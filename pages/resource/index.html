# HDR

# 分享四个免高质量的HDRI环境贴图网站

1，HDRI Havenhttps://hdri-haven.com/ (opens new window) 这个网站提供了各种高动态范围图像（HDRI），可以用于照明和环境渲染。它们的图像质量很高，提供了广泛的选择，适用于不同类型的项目。

2，Poly Havenhttps://polyhaven.com/ (opens new window) Poly Haven是另一个优秀的免费HDRI图像资源库。它提供了高质量、免费使用的HDRI图像，覆盖了各种环境和场景。

3，HDRMAPShttps://hdrmaps.com/ (opens new window) 这个网站提供了免费的HDR环境贴图，包括室内和室外场景。它们的图像质量很高，适用于渲染和照明。

4，Skyboxhttps://skybox.blockadelabs.com/ (opens new window) Skybox是一个在线的天空贴图生成器，提供了大量的免费天空贴图选择。你可以自定义不同的天空属性，比如云层、太阳位置等。

这些网站提供了免费的高质量HDRI图像，适用于各种3D渲染和建模项目。
